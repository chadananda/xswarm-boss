## ‚öôÔ∏è Configuration

In `~/.config/xswarm/config.toml`:

```toml

persona = "tars"
voice_enabled = true

[voice]
wake_word = "hey tars"

[voice.moshi]
pitch = 1.0
speed = 1.0
volume = 0.85
tone = "dry-pragmatic"

[theme.tars]
# TARS's unique personality settings
honesty_level = 90  # 0-100, how direct the feedback
humor_level = 75    # 0-100, frequency of dry jokes
mission_focus = true
self_aware_commentary = true
```
